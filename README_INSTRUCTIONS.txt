TYCOONS AI OPENAI GITHUB VERSION

Do not upload this ZIP directly to Netlify Drop for the AI function step.

Steps:
1. Download this ZIP.
2. Extract / unzip it on your computer.
3. Open GitHub repo: tycoons-ai-realestate.
4. Click Add file > Upload files.
5. Drag ALL extracted contents into GitHub:
   - index.html
   - styles.css
   - script.js
   - netlify.toml
   - package.json
   - netlify folder
6. Commit changes to the main branch.
7. Netlify should automatically deploy from GitHub.
8. Test https://tycoons-inv.de and search: ivilla garden in new cairo

Important:
- OPENAI_API_KEY must stay only in Netlify environment variables.
- Do not paste the OpenAI key into GitHub files.
